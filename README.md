# PREMIUM INTERIOR - B2B 상업용 인테리어 회사 웹사이트

> 기업 공간 전문 인테리어 파트너를 위한 프로페셔널 웹사이트

## 📋 프로젝트 개요

**PREMIUM INTERIOR**는 B2B 전문 상업용 인테리어 회사를 위한 완전한 반응형 웹사이트입니다. 
대기업, 프랜차이즈, 스타트업 등 기업 고객을 타겟으로 하는 전문적이고 신뢰감 있는 디자인으로 제작되었습니다.

### 🎯 프로젝트 목표
- B2B 기업 고객 유치 및 신뢰 구축
- 전문성과 실적을 강조하는 포트폴리오 전시
- 효율적인 견적 문의 프로세스 제공
- 모바일 최적화된 반응형 디자인

## ✨ 주요 기능

### 완료된 기능

#### 1. **메인 페이지 (index.html)**
- 풀스크린 히어로 섹션
- B2B 핵심 가치 제안 (4가지)
- 실적 통계 (200+ 프로젝트, 95% 재계약률)
- 서비스 미리보기
- 포트폴리오 미리보기 (3개)
- 주요 거래처 로고 섹션

#### 2. **회사소개 (about.html)**
- 대표 인사말
- 회사 연혁 (타임라인 디자인)
- 핵심 역량 4가지
- 실적 통계 상세
- 인증 및 수상 내역

#### 3. **서비스 소개 (services.html)**
- 4대 핵심 서비스
- 5단계 시공 프로세스 시각화

#### 4. **포트폴리오 (portfolio.html)**
- 실적 통계 바
- **카테고리 필터링 기능** (JavaScript)
- 9개 프로젝트 카드
- 호버 효과 및 상세 페이지 링크

#### 5. **프로젝트 상세 (project-detail.html)**
- 프로젝트 정보
- Before & After 이미지
- 이미지 갤러리

#### 6. **견적 문의 (contact.html)**
- B2B 맞춤 문의 폼
- **폼 검증 기능** (JavaScript)
- 연락처 정보
- FAQ 섹션

#### 7. **채용 정보 (recruiting.html)**
- 채용 공고
- 복리후생 소개

#### 8. **반응형 디자인**
- PC (1200px+)
- 태블릿 (768px - 1024px)
- 모바일 (< 768px)
